# Transactions

Manage and organize your transactions.

## Viewing Transactions

The Transactions page displays all imported transactions with:

- Date, merchant, description
- Amount (color-coded: green for income, red for expenses)
- Tags (bucket, occasion, account)
- Reconciliation status

## Filtering

### Quick Filters

One-click buttons for common queries:

- **This Month** / **Last Month** - Calendar month boundaries
- **This Year** / **YTD** - Year filters
- **Large** - Transactions above your dynamic threshold
- **Unreconciled** - Needs review

### Advanced Filters

- Text search (merchant, description, notes)
- Date range
- Amount range
- Account include/exclude
- Tag include/exclude
- Transfer status

## Tagging

Transactions can have multiple tags across namespaces:

| Namespace | Purpose | Example |
|-----------|---------|---------|
| `bucket` | Spending category | `groceries`, `dining` |
| `occasion` | Special events | `vacation-2024`, `birthday` |
| `account` | Source account | `checking`, `amex-gold` |

Only one bucket tag per transaction (auto-replaced on change).

## Bulk Operations

Select multiple transactions to:

- Apply the same bucket tag
- Mark as reconciled
- Mark as transfer
- Delete

## Saved Filters

Save complex filter combinations for reuse:

1. Configure filters
2. Click "Save Filter"
3. Name and optionally pin
4. Access from saved filters dropdown
